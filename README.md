# Account Bot (SQLite + aiogram)

Бот сохраняет аккаунты в формате `Ник:Пароль` в SQLite и по кнопке показывает список.
Пользоваться ботом может **только** владелец (по `ADMIN_ID`).

## Установка (Windows / PowerShell)

Перейдите в папку проекта:

```bash
cd C:\Users\fikus\account_bot
```

Создайте виртуальное окружение и активируйте:

```bash
py -m venv .venv
.venv\Scripts\activate
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Настройка

1) Создайте файл `.env` рядом с `bot.py` (можно скопировать из `.env.example`).

2) Заполните:

- `BOT_TOKEN` — токен вашего бота от @BotFather  
- `ADMIN_ID` — ваш Telegram user id (число)  
- `DB_PATH` — путь к базе (по умолчанию `data/bot.db`)

## Запуск

```bash
python bot.py
```

## Как пользоваться

- Добавить аккаунт: отправьте сообщение `Ник:Пароль`
- Показать список: кнопка **📋 Список аккаунтов**
- Удалить: кнопка **🗑 Удалить** под нужным аккаунтом

## Про "восстановление задач"

Бот хранит в таблице `tasks` pending-сообщения. При старте он читает их из SQLite и пытается отправить.
Это позволяет не терять важные уведомления при рестарте.
